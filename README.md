This repository stores the code for the research: <br><h1>The Impact of Image Resolution on the Detectability of Adversarial Examples</h1>
<br>
We divided into 4 parts to make it easier to follow.
<br>
01-Clean Model consist of scripts related training the clean model
<br>
02-AdvGenerate consist of scripts related to generate the adversarial examples
<br>
03-ExtractCharacteristics consist of scripts related to extract features (LayerPFS, LID, and JPEG Residual Analysis)
<br>
04-DetectionModel consist of script related to training the detection model
